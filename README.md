### Sobre o Projeto
O **StockMaster Pro** é uma solução Web completa para controlo de stock e inventário. Desenvolvido para oferecer uma experiência fluida tanto em Desktop como em Mobile, o sistema permite gerir produtos, quantidades e categorias com foco na segurança e integridade dos dados.

### Funcionalidades Principais
- **Autenticação Completa:** Sistema de Login e Registo de utilizadores.
- **CRUD de Produtos:** Criar, listar, editar e eliminar itens do inventário.
- **Gestão Inteligente:** Alertas visuais para stock baixo e cálculo de valor total.
- **Interface Responsiva:** Tabelas dinâmicas que se transformam em cards no mobile via Tailwind CSS.
- **Segurança:** Validação de inputs (Zod/Yup) e proteção contra vulnerabilidades comuns.

### Tech Stack
- **Frontend:** React.js (Vite)
- **Estilização:** Tailwind CSS
- **Navegação:** React Router Dom
- **Ícones:** Lucide React
